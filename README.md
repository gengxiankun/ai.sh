## Overview

ai.sh 是一键式 AI 个人站搭建工具。你将简历、想法和动态发布到站点上，访客无需翻阅页面，直接通过自然语言与 AI 对话，即可获取你的经历、作品、观点和最新分享。所有内容在线编辑，零代码部署，让你的个人站活起来。
